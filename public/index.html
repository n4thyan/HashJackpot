        plain English

        What is HashJackpot?

        It is a rare-hash leaderboard. One spin makes one SHA-256 hash. If it starts with zeroes, it scores. More zeroes means longer odds and a better rank.

        The Lotto comparison uses UK Lotto jackpot odds: 1 in 45,057,474. HashJackpot is not gambling. There is no money, crypto or prize pot.

        1. SpinYour browser rolls one hash.
        2. CountStarting zeroes set the score.
        3. PostGood scores can go on the leaderboard.
